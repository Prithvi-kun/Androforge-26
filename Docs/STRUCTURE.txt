---

## Component Responsibility Map

| File | Does what |
|------|-----------|
| `lib/supabase.js` | Single Supabase client + `submitRegistration()` + `fetchStats()` |
| `hooks/useRegistrationStats.js` | Subscribes to real-time INSERTs, returns `{ participantCount, teamCount, collegeCount, loading, error }` |
| `components/RegistrationForm.jsx` | Controlled form, client-side validation, calls `submitRegistration()`, shows success/error state |
| `components/LiveStats.jsx` | Consumes `useRegistrationStats`, renders 3 animated stat cards |

## Install Commands

```bash
# 1. Scaffold with Vite
npm create vite@latest androforge -- --template react
cd androforge

# 2. Install deps
npm install @supabase/supabase-js framer-motion

# 3. Install Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# 4. Start dev server
npm run dev
```

  content: ['./index.html', './src/**/*.{js,jsx,ts,tsx}'],
  theme: {

      fontFamily: {

      },
      animation: {
        'ping-slow': 'ping 2s cubic-bezier(0, 0, 0.2, 1) infinite',

  },
